# nl-shortener

URL redirects on a Nuclearn-owned domain.

Each `.html` file at the repo root is a redirect page. Requesting
`https://<pages-host>/<code>` (where `<code>.html` exists) loads a static
redirect page and forwards the browser to the target URL.

Codes are generated by [`shorten.py`](https://github.com/NuclearnAI/nl-shortener/blob/main/shorten.py)
(also lives in the Hermes `url-shortener` skill at
`~/.hermes/profiles/nuclearn/skills/productivity/url-shortener/scripts/shorten.py`).

Third-party shorteners are banned for Nuclearn URLs — see the
[data-locality convention in PM Corpus](https://github.com/NuclearnAI/) for
the rationale. This repo is the Nuclearn-owned replacement.

## Adding a redirect

```bash
python3 ~/.hermes/profiles/nuclearn/skills/productivity/url-shortener/scripts/shorten.py \
    --repo ~/code/nl-shortener \
    --pages-host <host> \
    "https://your-long-url.example.com/..."
```

Prints the resulting short URL. Commits and pushes automatically.
